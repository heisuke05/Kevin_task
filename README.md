# Kevin_task — TaskFlow

Notion風の個人タスク管理Webアプリです。バックエンドなし・登録不要・ブラウザだけで動作します。

## できること
- 今日やること / 期限超過 / 直近5件を自動表示
- 携帯でサッとメモ(Inbox)→ワンタップでタスク化
- プライベート / 仕事(道場ルーティーン・Web・X投稿・note・Substack)のタグ分け＋色分け
- チェックボックスでタスク完了
- リマインド通知（アプリを開いている間）＋ Googleカレンダー連携（ワンタップで予定追加）
- PC画面でガントチャート表示
- ダークモード基調（設定でライトモードに切替可）
- PWA対応（ホーム画面に追加してワンタップ起動、オフライン対応）

## 使い方（GitHub Pagesで公開する場合）
1. このリポジトリの Settings → Pages を開く
2. Source を「Deploy from a branch」、Branch を `main` / `/ (root)` に設定して Save
3. 数分後に `https://heisuke05.github.io/Kevin_task/` でアクセス可能になります
4. スマホでそのURLを開き、共有ボタン→「ホーム画面に追加」でアプリのように使えます

## データについて
データは各ブラウザの localStorage に保存されます（サーバーには送信されません）。
PCとスマホでデータを揃えたい場合は、設定画面の「エクスポート／インポート」で手動同期してください。
